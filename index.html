About Me

I am a DePaul University computer science bachelor currently seeking employment as a software engineer.

Work Experience

At DePaul I worked as an intern under professors Corin Pitcher, James Riley, and Radha Jagadeesan on a digital forensics project. I wrote and tested file system forensics software intended to benefit law enforcement agencies, which extracted and verified data from file systems so as to improve the credibility of evidence brought to court. My focus was on writing software for examining data stored in Microsoft's proprietary NTFS file system. Since NTFS has no publicly available specification, I had instead relied on information about the file system that had been previously gathered through reverse engineering as in File Sytem Forensic Analysis by Brian Carrier. I wrote software in a variety of programming languages for the project including Java, Scala, Coq, and OCaml.

Open Data

One of my interests is in creating applications from open government data. One example of this is the Chicago Police Map application I wrote using Ruby on Rails, Leaflet, and JQuery. The application takes crime data from the City of Chicago Data Portal, and plots criminal activity on a map of Chicago by police wards.

I have also developed a ClearPath Scala Client for the Chicago Police's ClearPath REST API, using AKKA and Spray.

Other Projects

One project I have developed is zipper. Zipper is a library for Scala that aims to simplify writing collections of files to zip archives, and extracting collections of files from zip archives. The library is designed to be easy to use as well as thread safe.

A project I developed in collaboration with other students at DePaul is JamTunes. JamTunes is a Web2Py application for social music sharing and discussion.

Problem Solving

In my spare time, I occasionally like to solve programming problems. I've solved several problems from past ACM programming competitions in Java, I've solved problems from Project Euler in Scala as well as in Ruby, and I've solved problems from CodeAbbey in Scala as well.
